# 页面布局

> 页面布局主要考察 HTML 以及 CSS 的功底，对页面布局的把控能力。

## 三栏布局

要求：高度已知，左右固定，中间自适应。

```html
<div class="container">
  <div class="main"></div>
  <div class="left"></div>
  <div class="right"></div>
</div>
```

### position 布局

```css
.container {
  position: relative;
}
```
